# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.4] - 2023-10-6

### Added 

- Activities (Chatting, listening to a story, searching for objects) now slow down the passage of time

### Changed

- Stories now give 8 points
- Searching for objects gives 8 points now if you answer everything correctly
- The limit of points in each chapter increased a little

### Fixed 

- Commercial break not working/running in english language (Incorrect translation in the commands)

## [0.1.3] (Only web version) - 2023-09-28

### Added

- Change Photo Sound

### Fixed

- Variable "data.FirstGame" was not working
- The static sound was not on loop
- Incorrect translation in the game

## [0.1.2] - 2023-09-27

### Added

- Fast forward the tape, when completing the necessary points and the show has not ended

### Changed

- Reduction of the effect of chromatic aberration
- The "Exit" button in the main menu should not be in the Web version
- Now clicking also advances the dialogs (PC and Web)

### Fixed

- The next dialog button did not disappear (Mobile)
- Commercial break not working/running
- The "Continue" button did not work
- "Delete data" did not work

## [0.1.1] - 2023-09-26

### Added

- UI improvements (mobile)
- Button to inflate another balloon (mobile)
- New action key for minigames

### Fixed

- Can't pause the game with the "Pause" button on the keyboard
- The progress bar in "Blow up balloons" did not fill
- Could not inflate another balloon
- Settings sliders were not working
- Image missing on first startup

### Changed

- Now to inflate another balloon is by pressing the second action key


## [0.1.0] - 2023-09-25

The first release
